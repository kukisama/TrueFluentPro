# 使用帮助

## 快速开始

1. 打开"配置中心"，在订阅管理中添加 Azure Speech 的 Key 与 Region ✅
2. 在端点管理中创建 AI 端点（可选择端点资料包快速配置）🔑
3. 选择输入来源（麦克风 / 回环）与设备 🎧
4. 点击"开始翻译"，等待字幕出现 ✨

## 输入源怎么选

- **麦克风**：适合会议、对话、现场收音
- **回环**：适合视频/系统声音，把"正在播放的声音"当输入
- **回环+麦克风**：同时采集系统声音与人声，适合需要双向收音的场景

## 字幕显示

- 需要时打开"浮动字幕"，方便边看边听
- 中间结果会实时刷新，最终结果会更稳定
- 浮动字幕支持调整透明度、字号与文字颜色

## 录音与历史

- 开启会话录音后，会先写 WAV，支持 MP3 自动转码（Windows）
- MP3 码率等参数可在配置中心调整
- 历史记录会按会话累积，方便回看整理

## AI 复盘与洞察

- 在 Review 模式下，可对会话内容进行 AI 分析
- 内置预设洞察：会议总结、知识点提取、客诉识别、行动项提取、情绪分析
- 支持自定义 Prompt，也可开启自动洞察模式持续输出分析
- 浮动洞察窗口：独立置顶窗口，实时查看分析结果，不遮挡主界面

## 批量字幕中心

- 在批量字幕中心，可对录音文件批量生成 SRT/VTT 字幕
- 需要配置 Azure Blob Storage 连接字符串（在配置中心 → 存储设置）
- 支持字幕分句方式、最大字符数、时长等参数调整
- 批任务支持队列管理、状态追踪与日志查看

## 创作工坊（对话）

- 点击顶部"💬 创作工坊"按钮进入 AI 对话模式
- 支持多会话管理，每个会话独立保存（SQLite）
- 对话中可触发联网搜索，AI 回复会自动标注引用来源（如 [1][2]）
- 支持拖拽上传文件与图片到对话中
- 可配置快捷短语与 Prompt 预设

## 创作工坊（媒体）

- 点击顶部"🎨 创作工坊"按钮进入媒体生成模式
- 支持 AI 图片生成（DALL·E 等模型）和视频生成（Sora 2 等模型）
- 图片生成可挂载参考图（最多 8 张），内置裁剪工具
- 视频生成支持图生视频，可配置分辨率与时长
- 每个任务保存为独立工作区，支持重命名、复制、删除

## 配置中心

配置中心包含 13 个模块，点击顶部"配置中心"按钮打开：

- **订阅管理**：添加/验证/测速多个 Azure Speech 订阅，支持中国区终结点
- **端点管理**：基于端点资料包创建 AI 端点，自动匹配 API 版本与能力策略
- **AI 配置**：设置 OpenAI 兼容 / Azure OpenAI 后端，支持 AAD（Entra ID）无密钥认证
- **网页搜索**：选择搜索引擎（Google / Bing / DuckDuckGo / 百度 / MCP），设置触发模式
- **图片生成**：DALL·E 模型选择、图片尺寸与样式
- **视频生成**：Sora 2 分辨率、时长等参数
- **洞察设置**：洞察 Prompt 自定义、预设按钮编辑、自动洞察触发
- **复盘设置**：复盘工作表预设、字号
- **录音设置**：开关录音、调整码率、自动删除 WAV 等
- **音频处理**：自动增益（AGC）、活动阈值、WebRTC 回音消除与降噪
- **存储设置**：Azure Blob Storage 连接字符串、容器与批目录
- **文本设置**：编辑器主题、字号、字体
- **导入导出**：一键导出/导入全量配置，方便迁移

## 常见问题

- **没有字幕**：确认订阅信息是否正确，网络是否正常
- **没声音/识别很少**：检查输入源与设备是否选对
- **回环无声音**：确认系统正在播放声音，并选择回环设备
- **AI 复盘/对话无输出**：检查配置中心中 AI 端点的 Endpoint 与 Key 是否正确
- **批量字幕失败**：确认 Azure Blob Storage 连接字符串已正确配置
- **联网搜索不生效**：检查配置中心 → 网页搜索中是否已启用并选择搜索引擎
- **AAD 认证失败**：确认 Azure.Identity 所需的登录凭据可用，检查租户配置
