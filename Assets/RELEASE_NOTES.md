# 译见 Pro v1.8.0

## 更新

- UI 全面升级：引入 Fluent Design，界面更现代清爽，操作更流畅
- 新增应用内一键更新：设置页可直接下载并安装新版本，带下载进度显示
- 右键菜单全面修复：字幕列表新增"跳转并播放"和"复制字幕"菜单，音频列表右键菜单恢复正常
- 修复暗色主题下文本编辑器出现大白块、译文颜色刺眼
- 浮动洞察窗口支持拖拽调整大小、字体缩放、右键切换背景色
- 双语视图原文/译文改为等比均分显示
- 通知提示改为右下角悬浮，不再顶开页面内容
- 设置页图片/视频生成参数恢复可编辑，修复重启后配置丢失
- 设置页与 Media Studio 的生成参数实时同步，不再需要手动切换刷新
- 修复 AI 洞察测试连接在 AAD 登录模式下报错，新增认证状态提示
- 修复洞察与复盘误提示"未配置 AI 服务"
- 修复"全部测试速度"按钮初始灰色不可点击
- 设置页导航重新整理，分区顺序更清晰
- 自动更新改为后台静默下载，完成后弹窗确认，不再频繁弹出通知
- 新增"启用自动更新"开关（设置 → 关于）

---


### 运行前提

本版本为 **FDD（Framework-dependent）** 发布，需要预先安装： [**.NET 10 Desktop Runtime**](https://dotnet.microsoft.com/download/dotnet/10.0)

### 使用方式

1. 下载对应架构的 zip 文件
2. 解压到任意目录
3. 运行 `TrueFluentPro.exe`
