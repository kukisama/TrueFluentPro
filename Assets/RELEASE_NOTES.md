# 译见 Pro v2.1.0

## 全新模块：听析中心（Audio Lab）

### 一站式音频智能分析
- 全新「听析中心」页面，支持 6 大分析维度：摘要、逐字稿、思维导图、洞察、研究、播客
- 基于 Whisper API 的真实 ASR 转录，配合 AI 摘要、思维导图生成、深度洞察
- 可变速播放（0.25x–4x），支持按片段导航跳转
- 可折叠文件面板，方便浏览本地音频文件

### 阶段预设系统
- 每个 AI 阶段支持自定义 Prompt、启用/禁用、显示/隐藏标签页、是否参与批处理
- 支持用户自定义阶段，动态生成对应标签页
- 阶段可见性随配置变化实时同步，无需重启
- 修改后的 Prompt 差异保存——未修改的 Prompt 升级时自动跟随代码默认值

### LLM 增强转录模式（预览）
- 使用 Azure Speech 增强模式，可选提示词辅助识别
- 播客脚本完成后自动触发 TTS 合成

### 任务执行审计
- 完整记录每次 AI 调用的 Token 消耗、模型名称、耗时、计费状态
- 调试模式：可查看每次调用的完整 Prompt 和响应内容

## 全新模块：音频任务队列

- 分类并发控制：转录任务（默认 10 并发）和 AI 任务（默认 5 并发）独立队列
- 任务监视器重新设计：左侧桶导航 + 阶段彩色标签 + 可排序列
- 右键菜单支持重试/取消单个任务
- 5 秒自动刷新，实时显示已用时间
- Token 统计面板：总执行数、计费次数、输入/输出 Token 汇总

## 图片生成大幅增强

### 多模型智能适配
- 支持 gpt-image-2 和 gpt-image-1.5 两代模型，界面根据所选模型自动调整可用选项
- gpt-image-1.5 可选透明背景（生成 PNG 抠图素材）和输入保真度（控制参考图细节保留程度）
- gpt-image-2 自动隐藏不支持的选项，提供自由画布选择任意分辨率
- 切换模型或切换会话时，参数面板即时刷新，无需手动调整

### 改图 V2：Responses API 多轮对话编辑
- 全新 V2 改图引擎，基于 Responses API（`/responses` + `image_generation` tool）
- 支持多轮对话式改图，通过 `previous_response_id` 维持上下文连贯
- Azure OpenAI 专属适配（`x-ms-oai-image-generation-deployment` 请求头）
- V2 不可用时自动回退至 V1 multipart 方式，用户无感知

### 创作工坊输入体验重做
- 参考 AskGo 设计语言，输入区重构为全宽文本框 + 底部工具栏
- 新增比例选择器（1:1 / 2:3 / 3:2 / 9:16 / 16:9 等），点选即用
- 附件、比例、发送/停止按钮整齐排列在底部，操作更直观
- 支持文件附件上传（图片 + 文本文件），支持拖放、Ctrl+V 粘贴

### 改图与跨轮记忆
- 聊天中提到「这张图」「改一下」「去掉背景」等意图时，自动关联最近一张图片参与改图，无需手动重新上传
- 参考图的 file_id 自动缓存，同一张图不会重复上传，节省等待时间

### 出图参数更完整
- 聊天模式也能读取配置面板的出图数量、分辨率、质量、格式
- 设置页新增「背景」和「保真度」默认参数

### 实时进度显示
- 图片生成全程可见进度：等待生成 → 下载中 → 保存中 → 完成，附带实时计时
- 聊天模式中 AI 触发图片生成时，气泡内显示「🎨 正在生成图片... 已等待 Xs」
- 创作工坊画布区新增进度浮层：呼吸动画 + 阶段描述 + 大字计时器 + 进度条

### 失败不再消失
- 生成失败的任务不再从结果列表中消失，改为显示红色错误卡片
- 错误信息自动翻译为中文友好提示（如内容策略拒绝、速率限制、超时等）

## 音频处理管线

### Microsoft Audio Stack (MAS) 集成
- 识别路径集成回音消除和噪声抑制

### WebRTC APM 实时预处理
- 回声消除（AEC）、噪声抑制（NS）、自动增益控制（AGC）、高通滤波（HPF）、预放大
- 统一音频管线架构：单路采集 → 插件预处理 → 共享录音与识别
- 设置页 WebRTC 插件自检按钮，一键验证模块状态
- 工具栏实时显示音频管线状态（启用模块、降噪分贝）

## 计费与成本追踪

- 新增独立计费账本，每次生成（含失败）都记录模型、参数、实际 token 消耗和估算成本
- 计费优先使用 API 返回的实际 token 数，精度显著提升
- 支持 gpt-image-2 和 gpt-image-1.5 各 9 档官方定价阶梯

## 画布尺寸选择器

- 设置页和图片模式均内嵌可视化画布选择器
- 拖拽右下角手柄自由选取尺寸，自动对齐 16px 网格
- 精简预设列表为 5 个核心值，画布拖出的自定义尺寸自动加入下拉框

## 数据存储全面升级

### SQLite 替代 JSON 文件
- 全新 SQLite 存储层（10 张表、11 个索引），彻底替代 JSON 文件持久化
- 启动时自动从旧 JSON 会话导入历史数据（一次性、幂等）
- 内容指纹去重，最小化磁盘写入
- 增量任务持久化：仅写入变化的数据
- 数据库自动 Schema 迁移（v1→v6）

### 媒体存储重新设计
- 按月分桶存储（`library/{type}/{yyyy}/{MM}/`），替代旧的按会话目录
- 不再创建空的工作区物理目录
- 「在文件夹中打开」自动定位高亮对应文件

## 启动与性能优化

- **原生预启动窗口**：点击应用图标后立即显示 Win32 原生闪屏，无需等待 Avalonia 初始化
- **ReadyToRun + TieredPGO**：显著减少 JIT 冷启动开销
- **AvaloniaEdit 样式懒加载**：首帧加载更快
- **配置 I/O 并行化**：多项配置并行读取
- **SQLite 写入优化**：写入次数减少 66%（466→156），运行时写入减少 92%（270→17）
- **SQLite 初始化移至后台线程**：UI 线程节省约 1.4 秒
- **会话 LRU 缓存**：仅保留 N 个会话在内存中，自动淘汰最久未用
- **会话切换异步加载**：支持取消，N+1 查询消除（4 条 SQL 批量加载）
- **创作工坊页面切换**：从销毁重建改为显示/隐藏，消除 re-layout
- **滚动性能**：移除逐帧视觉树遍历（3 次/帧 → 0 次）
- **视频帧提取**：用轻量 Vortice.MediaFoundation 替代 Microsoft.Windows.SDK.NET（发布包减小约 37MB）

## 界面体验改进

- 创作工坊会话切换：两阶段渐进式渲染（先显示最后 8 条消息，再补全其余）

- 配置加载成功提示 1 秒后自动关闭
- 字幕切割参数上限提升（最大字符 80→999，最大时长 15→99 秒）
- 审阅洞察：右键字幕 →「问 AI 关于这段内容」（发送到创作工坊）
