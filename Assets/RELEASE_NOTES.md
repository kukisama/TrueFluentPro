# 译见 Pro v2.0.3

## 更新

### 实时翻译
- 顶部工具栏新增**音频管线状态标签**：翻译中实时显示录制来源、WebRTC APM 模块状态、MAS 识别增强状态，悬停查看详细参数
- WebRTC 降噪增加**实时降噪量指标**（dB），直观证明音频处理插件确实在工作
- 录制来源（环回/麦克风/混合）切换后状态标签立即刷新
- 翻译过程中支持**实时切换源语言和目标语言**，无需停止翻译
- 修复"开始翻译"按钮悬停时颜色异常变化，现在保持红/绿纯色切换

### 复盘洞察
- 复盘模板新增**启用/禁用开关**，关闭后下次录音不处理该模板（不删除，随时可重新开启）
- 提示词输入框支持**聚焦展开多行**，离开焦点自动缩回，方便编辑长提示词

### 运行前提

本版本为 **FDD（Framework-dependent）** 发布，需要预先安装： [**.NET 10 Desktop Runtime**](https://dotnet.microsoft.com/download/dotnet/10.0)

### 使用方式

1. 下载对应架构的 zip 文件
2. 解压到任意目录
3. 运行 `TrueFluentPro.exe`
