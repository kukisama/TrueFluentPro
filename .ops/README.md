# .ops/ — 三角协作信箱

每个子目录 = 一个事件。有 `closure.md` = 已完成。

## 快速启动

```powershell
# 1. 对督工说第一句话（他会自动建目录开工）
copilot -p "你的需求描述" --agent "铁面督工" --model "claude-opus-4.6" --yolo

# 2. 自动循环（督工→工匠→参谋→督工...直到完成）
.\.ops\loop.ps1

# 3. 或手动逐步
copilot -p "继续" --agent "执行工匠" --yolo
copilot -p "继续" --agent "铁面督工" --yolo
```

## 角色

| Agent | 说明 | 触发 |
|-------|------|------|
| 铁面督工 | 决策+审查+简单任务独立完成 | 首句话 / "继续" |
| 执行工匠 | 按施工单实现 | "继续" |
| 毒舌参谋 | 攻击性审查（可选） | "继续" / 门禁 🐍 触发 |
