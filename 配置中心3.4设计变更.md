# 配置中心 3.4 设计变更记录

> 基于 `配置中心3.4设计.md` 的实施变更记录。

---

## 2026-03-04 — Phase 1：数据模型 + 配置迁移 + SettingsView ScrollSpy 框架

### 1. 新增数据模型（Models/AiEndpoint.cs）

| 类型 | 名称 | 说明 |
|------|------|------|
| `[Flags] enum` | `ModelCapability` | 模型能力标签：`Text=1`, `Image=2`, `Video=4` |
| `class` | `AiEndpoint` | 统一 AI 终结点：Id, Name, IsEnabled, ProviderType, BaseUrl, ApiKey, AuthMode, Models 列表 |
| `class` | `AiModelEntry` | 终结点下的模型定义：ModelId, DisplayName, DeploymentName, GroupName, Capabilities |
| `class` | `ModelReference` | 功能分区对模型的引用：EndpointId + ModelId |
| `class` | `ModelOption` | 下拉框显示类，格式 `终结点名 / 模型名` |

### 2. 配置根结构变化（Models/AzureSpeechConfig.cs）

- 新增 `List<AiEndpoint> Endpoints` 属性（统一终结点注册表）
- 新增 `GetAvailableModels(ModelCapability)` 方法（按能力筛选可用模型）
- 新增 `ResolveModel(ModelReference?)` 方法（解析引用为终结点+模型实例）

### 3. AiConfig 模型引用（Models/AiConfig.cs）

新增 4 个 `ModelReference?` 属性，与旧终结点字段并存以便渐进迁移：

- `InsightModelRef` — 洞察模型引用
- `SummaryModelRef` — 总结模型引用
- `QuickModelRef` — 快问模型引用
- `ReviewModelRef` — 复盘模型引用

### 4. MediaGenConfig 模型引用（Models/MediaGenConfig.cs）

新增 2 个 `ModelReference?` 属性：

- `ImageModelRef` — 图片生成模型引用
- `VideoModelRef` — 视频生成模型引用

### 5. 配置迁移逻辑（Services/ConfigurationService.cs）

新增 `MigrateEndpoints(AzureSpeechConfig)` 方法（internal static），在 `LoadConfigAsync` 中自动调用：

- 检测 `Endpoints` 为空时执行一次迁移
- 从 `AiConfig.ApiEndpoint` 迁移 → 创建"默认文字"终结点 + Text 模型
- 从 `MediaGenConfig.ImageApiEndpoint` 迁移 → 创建或合并"默认图片"终结点 + Image 模型
- 从 `MediaGenConfig.VideoApiEndpoint`（或复用 Image 端点）迁移 → 创建或合并"默认视频"终结点 + Video 模型
- 自动生成 `ModelReference` 指向迁移后的终结点+模型

### 6. SettingsView ScrollSpy 布局（Views/SettingsView.axaml + .cs）

**布局**：左侧固定导航 (180px ListBox) + 右侧瀑布流 (ScrollViewer)

**12 个分区**：

| # | Name | 分区标题 | 状态 |
|---|------|----------|------|
| 1 | `Section_Subscription` | 订阅管理 | 已迁入基础内容 |
| 2 | `Section_Audio` | 音频设备 | 已迁入基础内容 |
| 3 | `Section_Storage` | 录音与存储 | 占位 |
| 4 | `Section_Endpoints` | AI 终结点管理 | 占位 |
| 5 | `Section_Recognition` | 语音识别 | 占位 |
| 6 | `Section_Text` | 字幕与文本 | 占位 |
| 7 | `Section_Insight` | AI 洞察 | 占位 |
| 8 | `Section_Review` | 复盘预设 | 占位 |
| 9 | `Section_ImageGen` | 图片生成 | 占位 |
| 10 | `Section_VideoGen` | 视频生成 | 占位 |
| 11 | `Section_Links` | 外部链接 | 已迁入基础内容 |
| 12 | `Section_About` | 帮助与关于 | 已迁入基础内容 |

**ScrollSpy 实现**：
- 点击左侧导航项 → `BringIntoView()` 滚动到对应分区
- 右侧滚动 → 检测当前可见分区 → 自动高亮左侧对应导航项
- `_suppressScrollSync` 防止双向触发循环
