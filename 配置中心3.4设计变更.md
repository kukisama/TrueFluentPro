## 2026-03-04 — Phase 1：数据模型 + 配置迁移 + SettingsView ScrollSpy 框架

### 1. 新增数据模型（Models/AiEndpoint.cs）

| 类型 | 名称 | 说明 |
|------|------|------|
| `[Flags] enum` | `ModelCapability` | 模型能力标签：`Text=1`, `Image=2`, `Video=4` |
| `class` | `AiEndpoint` | 统一 AI 终结点：Id, Name, IsEnabled, ProviderType, BaseUrl, ApiKey, AuthMode, Models 列表 |
| `class` | `AiModelEntry` | 终结点下的模型定义：ModelId, DisplayName, DeploymentName, GroupName, Capabilities |
| `class` | `ModelReference` | 功能分区对模型的引用：EndpointId + ModelId |
| `class` | `ModelOption` | 下拉框显示类，格式 `终结点名 / 模型名` |

### 2. 配置根结构变化（Models/AzureSpeechConfig.cs）

- 新增 `List<AiEndpoint> Endpoints` 属性（统一终结点注册表）
- 新增 `GetAvailableModels(ModelCapability)` 方法（按能力筛选可用模型）
- 新增 `ResolveModel(ModelReference?)` 方法（解析引用为终结点+模型实例）

### 3. AiConfig 模型引用（Models/AiConfig.cs）

新增 4 个 `ModelReference?` 属性，与旧终结点字段并存以便渐进迁移：

- `InsightModelRef` — 洞察模型引用
- `SummaryModelRef` — 总结模型引用
- `QuickModelRef` — 快问模型引用
- `ReviewModelRef` — 复盘模型引用

### 4. MediaGenConfig 模型引用（Models/MediaGenConfig.cs）

新增 2 个 `ModelReference?` 属性：

- `ImageModelRef` — 图片生成模型引用
- `VideoModelRef` — 视频生成模型引用

### 5. 配置迁移逻辑（Services/ConfigurationService.cs）

新增 `MigrateEndpoints(AzureSpeechConfig)` 方法（internal static），在 `LoadConfigAsync` 中自动调用：

- 检测 `Endpoints` 为空时执行一次迁移
- 从 `AiConfig.ApiEndpoint` 迁移 → 创建"默认文字"终结点 + Text 模型
- 从 `MediaGenConfig.ImageApiEndpoint` 迁移 → 创建或合并"默认图片"终结点 + Image 模型
- 从 `MediaGenConfig.VideoApiEndpoint`（或复用 Image 端点）迁移 → 创建或合并"默认视频"终结点 + Video 模型
- 自动生成 `ModelReference` 指向迁移后的终结点+模型

### 6. SettingsView ScrollSpy 布局（Views/SettingsView.axaml + .cs）

**布局**：左侧固定导航 (180px ListBox) + 右侧瀑布流 (ScrollViewer)

**12 个分区**：

| # | Name | 分区标题 | 状态 |
|---|------|----------|------|
| 1 | `Section_Subscription` | 订阅管理 | 已迁入基础内容 |
| 2 | `Section_Audio` | 音频设备 | 已迁入基础内容 |
| 3 | `Section_Storage` | 录音与存储 | 占位 |
| 4 | `Section_Endpoints` | AI 终结点管理 | 占位 |
| 5 | `Section_Recognition` | 语音识别 | 占位 |
| 6 | `Section_Text` | 字幕与文本 | 占位 |
| 7 | `Section_Insight` | AI 洞察 | 占位 |
| 8 | `Section_Review` | 复盘预设 | 占位 |
| 9 | `Section_ImageGen` | 图片生成 | 占位 |
| 10 | `Section_VideoGen` | 视频生成 | 占位 |
| 11 | `Section_Links` | 外部链接 | 已迁入基础内容 |

- 点击左侧导航项 → `BringIntoView()` 滚动到对应分区

## 2026-03-05 — Phase 2：SettingsViewModel + 完整 UI 迁移 + 删除 ConfigCenterView

### 1. 新增 SettingsViewModel（ViewModels/SettingsViewModel.cs）

从 ConfigCenterView.axaml.cs 的 1,317 行事件驱动代码中提取业务逻辑到 `SettingsViewModel`：

**职责划分**：

| 模块 | 说明 |
|------|------|
| 订阅管理 | CRUD + 验证（AddSubscriptionCommand, UpdateSubscriptionCommand, DeleteSubscriptionCommand, TestSubscriptionCommand, TestAllSubscriptionsCommand） |
| 终结点管理 | CRUD（AddEndpointCommand, RemoveEndpointCommand）+ 模型注册（AddModelToSelectedEndpoint, RemoveModelFromSelectedEndpoint） |
| 录音与存储 | EnableRecording, RecordingMp3BitrateKbps, SessionDirectory, BatchStorage 验证等 |
| 语音识别 | FilterModalParticles, MaxHistoryItems, RealtimeMaxLength, ChunkDurationMs, 静音超时, 无回显重连等 |
| 字幕与文本 | ExportSrt, ExportVtt, DefaultFontSize |
| AI 基础配置 | AiProviderTypeIndex, AiApiEndpoint, AiApiKey, 模型名/部署名, AAD 认证字段 |
| AI 洞察 | 系统提示词, 预设按钮编辑 (PresetButtons), 测试连接 |
| 复盘预设 | 复盘 Sheet 编辑 (ReviewSheets), 系统提示词 |
| 模型引用 | TextModels/ImageModels/VideoModels 筛选 + SelectedInsightModel/SelectedQuickModel/SelectedSummaryModel/SelectedReviewModel/SelectedImageModel/SelectedVideoModel |

## 2026-03-05 — Phase 2 补丁：配置中心六项问题修复

### 问题清单与修复

| # | 问题 | 修复 |
|---|------|------|
| 1 | 终结点配置"添加模型"按钮无反应，已有模型无法编辑 | `SelectedEndpointModels` 改为返回 `.ToList()` 快照确保 UI 刷新；模型列表改为内联编辑（ModelId/DisplayName/DeploymentName/Capabilities 均可直接编辑） |
| 2 | 删除模型条目不立即生效，需切换选项卡 | 同上，`.ToList()` 快照使 `OnPropertyChanged` 真正触发 ItemsControl 重绑定 |
| 3 | 导航点击近距离分区不跳转 | `OnNavSectionChanged` 从 `BringIntoView()` 改为计算目标在 ScrollViewer 内容中的绝对偏移量并直接设置 `SettingsScroller.Offset` |
| 4 | 复盘/图片/视频模型默认为空 | `SelectModelOption` 在引用为 null 或未匹配时自动回退到列表第一个可用模型 |
| 5 | 复盘洞察模版丢失 | `LoadFromConfig` 检测 `ai.ReviewSheets` 为空时自动回退到 `new AiConfig().ReviewSheets` 默认模版（总结/情绪/客户顾虑/知识点 4 项） |
| 6 | 终结点缺少 AAD 认证 | 终结点详情新增认证方式 ComboBox（ApiKey/AAD）、Tenant ID、Client ID 字段；新增 `SelectedEndpointAuthMode` / `IsSelectedEndpointAad` 属性 |

### 变更文件

**ViewModels/SettingsViewModel.cs**：
- `SelectedEndpointModels` → `SelectedEndpoint?.Models?.ToList()`（快照，修复 UI 不刷新）
- 新增 `SelectedEndpointAuthMode` 属性（get/set 代理到 `SelectedEndpoint.AuthMode`）
- 新增 `IsSelectedEndpointAad` 计算属性
- 新增 `NotifyModelChanged()` 方法（内联编辑后触发同步保存）
- `SelectModelOption` 增加 fallback 逻辑：引用为 null 或未匹配时选择第一个可用模型
- `LoadFromConfig` 中 ReviewSheets 为空时回退到默认模版

**Views/SettingsView.axaml**：
- 终结点详情区新增认证方式 ComboBox + AAD 字段（Tenant ID / Client ID）
- 模型列表从只读 TextBlock 改为内联编辑 TextBox（ModelId / DisplayName / DeploymentName）+ 能力 ComboBox

**Views/SettingsView.axaml.cs**：
- `OnNavSectionChanged` 改为直接计算偏移量滚动
- 新增 `ModelField_LostFocus` 事件处理（编辑模型字段后触发保存）
- 新增 `ModelCapability_SelectionChanged` 事件处理（能力下拉变化后同步模型）
